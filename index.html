Digital Foot Print

Active VS Passive -> Consequences

Active is where you submit somehting like a social media account or aa youtube channel, so you know it happend because you submitted it yourself. Passive is done without your consent where they scrape the web for stuff related to and that could be traced to you, this is done without consent. The consequences of this are that both passive and active info can be used against you in cyber vetting, Hacking, and sharing of your private info.

IP Grabber

Most every device that connects to the internet has two different IP codes, one that is public and another is private, some of the best IP grabbers can take your private ip in just one click. The IP code consists of numerical values witch can be deciphered in the location. Grabbers are used to steal these numbers and other web based utilites are used to decyther and abuse the ip, some bad use cases would be to steal somebodies location or launch a dos, or denial of service attack.

Ad Blockers

Another way to stay safe on the internet is using an ad-blocker these work in a variety of ways but the best ones that are run on a raspberry pi for the whole network block the server adressed for the ads so that they could never load in the first place. In our modern world where ads rule the internet its shockingly easy to get an ad blocker you can ust look up ad blocker extension and add the first one to chrome or any other browser that you use. Some websites like youtube are beginning to fight back against ad-blockers by blocking users witht hem from using the website temporarily.
